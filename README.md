# 👗 舞衣 - 智能电子衣橱

> 一个基于 Vue 3 + TypeScript 的智能电子衣橱应用，帮你管理衣物、推荐穿搭、虚拟试衣。

---

## ✨ 功能特性

- 📸 **衣物管理** - 拍照上传衣物，自动识别分类
- 🎨 **穿搭推荐** - 智能推荐今日穿搭，支持多种风格
- 👤 **虚拟试衣** - 3D 模特展示，调整身材数据看效果
- ❤️ **收藏穿搭** - 收藏喜欢的搭配方案
- 📊 **衣橱统计** - 查看衣物数量、品类分布

## 📦 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 打包
npm run build
```

## 🛠️ 技术栈

- Vue 3 + TypeScript
- Vite
- Pinia (状态管理)
- uni-app (跨平台)
- Three.js (3D 模特)
